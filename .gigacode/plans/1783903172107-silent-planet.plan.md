# План: Добавление SMTP-настроек в CloudSettingsScreen

## Цель
Добавить поля для ввода SMTP-конфигурации (хост, порт, username, пароль) в CloudSettingsScreen.kt для настройки автоматической отправки email через Yandex SMTP.

## Шаги реализации

### 1. Добавить импорт PasswordVisualTransformation
- **Файл**: `app/src/main/java/com/example/ohrana/CloudSettingsScreen.kt`
- **Изменение**: Добавить `import androidx.compose.ui.text.input.PasswordVisualTransformation`

### 2. Добавить состояние для SMTP полей
- **Файл**: `app/src/main/java/com/example/ohrana/CloudSettingsScreen.kt`
- **Место**: После переменных smsPhone, email, telegram
- **Изменение**: Добавить переменные:
  ```kotlin
  var smtpHost by remember { mutableStateOf(prefsManager.getSmtpHost() ?: "smtp.yandex.ru") }
  var smtpPort by remember { mutableStateOf(prefsManager.getSmtpPort().toString()) }
  var smtpUsername by remember { mutableStateOf(prefsManager.getSmtpUsername() ?: "") }
  var smtpPassword by remember { mutableStateOf(prefsManager.getSmtpPassword() ?: "") }
  ```

### 3. Загрузить SMTP значения при инициализации
- **Файл**: `app/src/main/java/com/example/ohrana/CloudSettingsScreen.kt`
- **Место**: Внутри LaunchedEffect(cloudManager, prefsManager)
- **Изменение**: Добавить строки загрузки SMTP значений из SharedPrefsManager

### 4. Заменить информационную карточку SMTP на поля ввода
- **Файл**: `app/src/main/java/com/example/ohrana/CloudSettingsScreen.kt`
- **Место**: Ветка SharedPrefsManager.LinkAction.EMAIL_SMTP
- **Изменение**: Заменить Card с текстом на 4 OutlinedTextField:
  - smtpHost (хост)
  - smtpPort (порт, KeyboardType.Number)
  - smtpUsername (email, KeyboardType.Email)
  - smtpPassword (пароль, PasswordVisualTransformation)

### 5. Обновить функцию saveSettings
- **Файл**: `app/src/main/java/com/example/ohrana/CloudSettingsScreen.kt`
- **Изменения**:
  - Добавить 4 новых параметра: smtpHost, smtpPort, smtpUsername, smtpPassword
  - Внутри вызвать методы prefsManager.saveSmtpHost(), saveSmtpPort(), saveSmtpUsername(), saveSmtpPassword()
  - Обновить вызов функции saveSettings в Button.onClick

### 6. Проверить компиляцию
- Выполнить сборку проекта через Gradle

## Ожидаемый результат
- В интерфейсе настроек появятся поля для ввода SMTP-конфигурации при выборе действия "Email (SMTP)"
- Поля будут сохраняться в SharedPrefsManager
- При повторном открытии настроек значения будут загружаться из памяти
- Проект будет компилироваться без ошибок
