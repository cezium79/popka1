# Система централизованного управления дизайном UI-элементов

Этот файл описывает, как использовать централизованную систему дизайнов кнопок, карточек и свитчей.

## Структура файлов

- `ButtonDesigns.kt` - Дизайны кнопок (10 вариантов)
- `CardAndSwitchDesigns.kt` - Дизайны карточек (8 вариантов) и свитчей (5 вариантов)
- `OhranaButtons.kt` - Компоненты кнопок с поддержкой дизайнов
- `OhranaCards.kt` - Компоненты карточек с поддержкой дизайнов
- `OhranaSwitches.kt` - Компоненты свитчей с поддержкой дизайнов

## Как использовать

### Кнопки

#### 1. Использование по ID дизайна (рекомендуется)

```kotlin
// Вставьте кнопку с дизайном №3
OhranaButton(
    text = "Нажми меня",
    onClick = { /* обработчик */ },
    designId = 3  // Использует DESIGN_3_OUTLINED
)
```

#### 2. Использование по названию

```kotlin
OhranaButton(
    text = "Нажми меня",
    onClick = { /* обработчик */ },
    designId = getButtonDesignByName("Outlined")?.id
)
```

#### 3. Полный список дизайнов кнопок

| ID | Название | Описание | Цвет |
|----|----------|----------|------|
| 1 | Standard | Стандартная кнопка | Фиолетовый (#6200EE) |
| 2 | Contained | Контейнерная кнопка | Фиолетовый (#6200EE) |
| 3 | Outlined | Обеченная кнопка | Прозрачный + фиолетовый текст |
| 4 | Text | Текстовая кнопка | Прозрачный |
| 5 | Elevated | Кнопка с тенями (8dp) | Фиолетовый (#6200EE) |
| 6 | Destructive | Красная кнопка (опасные действия) | Красный (#B00020) |
| 7 | Success | Зеленая кнопка (успех) | Зеленый (#4CAF50) |
| 8 | Info | Синяя кнопка (информация) | Синий (#2196F3) |
| 9 | Gold | Золотая кнопка (премиум) | Золотой (#FFD700) |
| 10 | Dark | Черная кнопка | Черный (#1A1A1A) |

#### 4. Переопределение параметров дизайна

```kotlin
OhranaButton(
    text = "Нажми меня",
    onClick = { /* обработчик */ },
    designId = 3,  // Использует дизайн №3
    colors = ButtonDefaults.buttonColors(
        containerColor = Color.Red  // Но переопределяет цвет контейнера
    )
)
```

### Карточки

```kotlin
@Composable
fun OhranaCard(
    modifier: Modifier = Modifier,
    designId: Int? = null,
    onClick: (() -> Unit)? = null,
    content: @Composable () -> Unit
) {
    // Реализация использует дизайн из CardDesigns
}

// Использование:
OhranaCard(designId = 2) {
    Text("Содержимое карточки")
}
```

### Свитчи

```kotlin
// Использование свитча с дизайном №2
OhranaSwitch(
    checked = isChecked,
    onCheckedChange = { isChecked = it },
    designId = 2  // Яркий дизайн (зеленый)
)
```

## Преимущества системы

1. **Единообразие** - Все элементы одного типа используют единый стиль
2. **Легкая настройка** - Измените дизайн в одном месте и он обновится везде
3. **Простота использования** - Укажите `designId` и готово
4. **Гибкость** - Можно переопределить отдельные параметры при необходимости

## Как добавить новый дизайн

1. Откройте `ButtonDesigns.kt` или `CardAndSwitchDesigns.kt`
2. Добавьте новый дизайн в объект `ButtonDesigns`/`CardDesigns`/`SwitchDesigns`
3. Добавьте обработку в функции `get...DesignById()` и `get...DesignByName()`
4. Обновите список в `getAll...Designs()`

## Текущие дизайны

### Кнопки (10 штук)
1. Standard - Стандартная
2. Contained - Контейнерная
3. Outlined - Обеченная (рекомендуется для основных действий)
4. Text - Текстовая
5. Elevated - Сильные тени
6. Destructive - Красная (опасные действия)
7. Success - Зеленая (успех)
8. Info - Синяя (информация)
9. Gold - Золотая (премиум)
10. Dark - Черная

### Карточки (8 штук)
1. Standard - Стандартная (8dp углы)
2. Elevated - С тенями (12dp углы)
3. Outlined - Обеченная
4. Accent - Акцентный цвет
5. Dark - Темная
6. Light - Светлая
7. Rounded - Закругленная (20dp)
8. Circle - Круглая

### Свитчи (5 штук)
1. Standard - Стандартный (синий)
2. Bright - Яркий (зеленый)
3. Dark - Темный
4. Gold - Золотой
5. Destructive - Красный
